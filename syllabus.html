Course Description

This course introduces the basic principles of network security. Students will be introduced to computer network vulnerabilities and threats and how to safeguard computer networks from those vulnerabilities and threats. This course will expose students to network security planning, network security technology, network security organization and the legal and ethical issues associated with network security. This course is designed to prepare successful students for the CompTIA Security+ certification.

Major Course Concepts

A. Network Security Basics

  • Security Definitions, Network security principles and objectives
  • Network security architecture, model, and layers
  • Network threats, confidentiality, integrity and availability
  • Accountability, auditability, and non-repudiation

B. Network Security Planning

  • Network risk management and network security policy
  • Continuity of operations and security maintenance
  • Policy, procedures, standards, and guidelines

C. Network Security Technology

  • Cryptography, virtual private networks, firewalls and intrusion detection systems
  • Scanning and analysis tools and access controls

D. Network Security Organization

  • The organization and security
  • Personnel and security
  • The Computer Incident Response Team (CIRT)

E. Legal, Privacy, and Ethical Issues

  • US Laws and Network Security
  • Privacy and network security
  • Ethics and Network Security
