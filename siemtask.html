Assignment Overview

You are a Security Operations Center (SOC) analyst for FreshFlow Systems, a food processing and distribution facility in Chicago, IL. Your task is to use the FreshFlow Systems SIEM Simulator to generate, analyze, and interpret various security and operational events. This assignment will test your ability to operate SIEM tools and understand industrial control system security.

Pre-Lab Instructions

  1. Open the FreshFlow Systems SIEM Simulator in a separate tab/window
  2. Familiarize yourself with the control panel and log display
  3. Note the equipment profile information in the upper left panel
  4. Ensure the timestamp is updating correctly
  5. Return to this assignment page to begin the tasks

Task 1: System Event Analysis

10 Points

Generate and analyze system-level events

Instructions:

  1. Set Log Event Type to "System Events"
  2. Set Severity Level to "INFO"
  3. Select Equipment: "SCADA-01 (SCADA Master Station)"
  4. Generate 5 different log entries by clicking "Generate Log Entry" five times
  5. Observe the log format and content
